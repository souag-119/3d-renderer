# Render3D Studio — محرك رندر Path Tracing

تطبيق ويب كامل لرندر مشاهد 3D بتقنية Path Tracing، يعمل بالكامل داخل المتصفح بدون أي تثبيت أو اتصال إنترنت.

## طريقة التشغيل

هذا التطبيق يحتاج تشغيله عبر خادم محلي (وليس فتح index.html مباشرة كملف) بسبب متطلبات المتصفح الأمنية لبعض الميزات. الطريقة الأسهل:

### الطريقة 1: عبر Python (إن كان مثبتاً على جهازك)
1. فك ضغط الملف.
2. افتح Terminal / موجه الأوامر داخل مجلد المشروع.
3. نفّذ:
   ```
   python3 -m http.server 8080
   ```
4. افتح المتصفح على: `http://localhost:8080`

### الطريقة 2: عبر امتداد Live Server في VS Code
1. افتح مجلد المشروع في VS Code.
2. ثبّت امتداد "Live Server".
3. اضغط بزر الفأرة الأيمن على index.html واختر "Open with Live Server".

### الطريقة 3: أي استضافة استاتيكية
ارفع الملفات كما هي إلى أي استضافة (GitHub Pages، Netlify، Vercel، إلخ) — لا حاجة لأي backend أو بناء (build).

## المتطلبات
- متصفح حديث يدعم **WebGL2** (Chrome، Edge، Firefox، Safari الحديثة).
- يُفضّل جهاز به معالج رسومي (GPU) للحصول على أداء سلس أثناء المعاينة الحية.

## الميزات
- **محرك Path Tracing** حقيقي يعمل بالكامل على GPU عبر WebGL2 Fragment Shader.
- **تقسيم الشاشة إلى بلاطات (Tiles)** قابلة للتخصيص لتوزيع حمل الرندر وتحديث الصورة تدريجياً.
- **Denoiser اختياري** (فلتر ثنائي الجانب يحافظ على الحواف) مع تحكم بالقوة.
- **عناصر أساسية**: مكعب، كرة، مستوى، اسطوانة، مخروط، حلقة (Torus).
- **إضاءات**: نقطية، شمسية (اتجاهية)، مساحية — بألوان وشدة ونعومة ظل قابلة للتعديل.
- **نظام متريال PBR كامل**: لون أساسي، Metallic، Roughness، IOR، Transmission (شفافية/زجاج)، Emission (إصدار ضوئي) — مع مكتبة ماتريالات، إنشاء جديد، نسخ، وزر "تطبيق".
- **كاميرا احترافية**: FOV، عمق المجال (Depth of Field) مع مسافة تركيز وفتحة عدسة، بيئة سماء متدرجة قابلة للتخصيص.
- **نافذة إخراج (Render Output)** منفصلة: اختيار الدقة (حتى Full HD أو دقة مخصصة)، عدد العينات، شريط تقدم، وحفظ الصورة كـ PNG.
- **Gizmo تفاعلي**: نقل / تدوير / تحجيم العناصر مباشرة في المشهد، مع أزرار نسخ وحذف سريعة.
- **واجهة عربية بالكامل (RTL)** مصممة خفيفة الأداء ومتجاوبة مع الجوال وسطح المكتب.

## طريقة الاستخدام السريعة
1. أضف عناصر أو إضاءات من اللوحة اليمنى (أو اليسرى حسب اتجاه الشاشة).
2. اضغط على العنصر في شجرة المشهد لتحديده، ثم استخدم Gizmo لتحريكه في الفيوبورت.
3. من تبويب "المتريال" اختر أو أنشئ متريال، عدّل خصائصه، ثم اضغط "تطبيق".
4. من تبويب "الكاميرا" فعّل عمق المجال واضبط الإعدادات حسب رغبتك.
5. اضغط "نافذة الإخراج" لاختيار الدقة النهائية وعدد العينات، ثم "بدء الرندر"، وأخيراً "حفظ الصورة".

## ملاحظة على الأداء
الرندر يتم بالكامل عبر المعالج الرسومي (GPU) في متصفحك. على الأجهزة الضعيفة أو بدون تسريع رسومي (Software Rendering)، قد تكون المعاينة الحية بطيئة نسبياً — هذا طبيعي ومتوقع لتقنية Path Tracing، وهو بالضبط سبب استخدام آلية التقسيم إلى بلاطات (Tiles) بدل تجميد الواجهة أثناء الحساب.
